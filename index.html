What is CSS Grid

CSS grid is a layout tool for websites utilizing html and css. Websites can be easily organized and aligned into columns and rows.

Initial Step

To turn any element into a grid container, you need to create a grid in CSS. This is done by using the property display and setting it to grid. For instance, for the div with the id container, this is what it would look like:

.container {
          display: grid;
        }

At this point you can utilize all of the other tools associated with CSS Grid.

Grid Columns

After the grid element is created, you can define the structure of your grid in several ways. To organize your grid into columns, use the grid-template-column property, like this:

.container {
          display: grid;
          grid-template-columns: 50px 50px;
        }

This will give your grid two columns 50px wide. To increase the number of columns, simply add extra widths. To create 4 columns 100px wide, the code would look like this: 

.container {
          display: grid;
          grid-template-columns: 100px 100px 100px 100px;
        }
Grid Rows

Organizing your grid into rows is as easy as organizing it into columns. Use the grid-template-rows property and define the widths for the number of rows you would like. To create 3 rows, 20px each, it would look like this:

.container {
          display: grid;
          grid-template-rows: 20px 20px 20px;
        }
Columns and Rows

Have just columns or just rows in a grid is too simple for most websites. Both are usually incorporated to allow for several sections. To create a grid with 3 columns and 3 rows, this is what you would code:

.container {
          display: grid;
          grid-template-rows: 50px 50px 50px;
          grid-template-columns: 50px 50px 50px;
        }

This would create a grid 3 cells across and 3 cells down.

Cell Measurements

When creating columns and rows you are not limited to using the pixel measurement. The cells created with the rows and columns can be measured in both relative and absolute measurements. Here is a list of some of the measurements that can be used:

  • px - measures the cells into pixels
  • em - measures the cells by ems
  • fr - sets the column or row to a fraction of the available space
  • auto: sets the column or row to the width or height of its content automatically
  • %: adjusts the column or row to the percent width of its container

These measurements can be used at the same time, for instance:

.container {
          display: grid;
          grid-template-rows: 50px 50px 50px;
          grid-template-columns: auto 50px 10% 2fr 1fr;
        }

This would create a grid with 3 uniform rows, each 50px. It would also create 5 columns. The first column is as wide as its content, the second column is 50px, the third column is 10% of its container, and for the last two columns; the remaining space is divided into three sections, two are allocated for the fourth column, and one for the fifth.

Grid Gaps

To place gaps between the cells, 3 properties can be used - grid-row-gap, grid-column-gap, and grid-gap, followed by the width of the desired gap. In the case of the property grid-gap, if only one value is used, it will set the gap for both columns and rows, but if two values are used, the first will set the gap between the rows and the second will set the gap between the columns. For instance:


          grid-row-gap: 5px //sets the gap between the rows to 5px
          grid-column-gap: 5px //sets the gap between the rows to 5px
          grid-gap: 50px //sets the gap between the rows and columns to 5px
          grid-gap: 5px 10px //sets the gap between the rows to 5px and the columns to 10px
        }
Spacing and Aligning

Now that the columns, rows, gaps, and cells have been formatted, there are many options. You can align and justify content within the cells, you can create cells that are contained within several cells, and you can create templates that can be manipulated.
